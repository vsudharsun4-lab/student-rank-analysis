## Key Differences

| Feature | OLD Dashboard | NEW Dashboard |
|---------|---|---|
| **Layout** | Cluttered, 1-page | Clean hub + 5 focused pages |
| **Navigation** | Scroll endlessly | Click cards to navigate |
| **Load Time** | Slow (all data) | Fast (focused data) |
| **Usability** | Confusing | Intuitive |
| **Design** | Generic | Modern & professional |
| **Information Density** | Overwhelming | Digestible |
| **Task Focus** | Mixed purposes | One purpose per page |
| **Mobile Experience** | Poor | Responsive & clean |

### OLD: User looking for intervention candidates
```

3. Find "Intervention Candidates" somewhere in the middle
```

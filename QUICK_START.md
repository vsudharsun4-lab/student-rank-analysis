## 🐛 Troubleshooting

### Issue: Pages not loading?
**Solution:** Make sure Flask is running (`python app.py`)

### Issue: Can't see data?
**Solution:** Make sure you're logged in with valid credentials

### Issue: Charts not showing?
**Solution:** Check that Chart.js library loads (should be automatic)

### Issue: Styles look weird?
**Solution:** Hard refresh browser (Ctrl+F5 or Cmd+Shift+R)
